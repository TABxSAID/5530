## Overview

This repository contains the complete final project notebook for CS-5530 Principles of Data Science (Fall 2025).  
The notebook demonstrates core data science concepts using a **reproducible, personalized** workflow.

## Sections included

Part 0 – Reproducibility & personalization  
Part A – Ethics & misleading visualizations  
Part B – Data preprocessing & outlier handling  
Part C – Model training (KNN & Decision Tree)  
Part D – Cross-validation & bootstrap confidence intervals  
Part E – Simulation & Law of Large Numbers  
Part F – Regex + TF-IDF sentiment analysis
