# Diabetes Sampling and Bootstrap

### Tasks completed:
a) Random sample of 25, finding the mean Glucose and highest Glucose values of this sample and comparing the results using charts.   
b) Finding the 98th percentile of BMI of your sample and the population and comparing the results using charts.   
c) Using bootstrap to create 500 samples of 150 observation each from the population and finding the average mean, standard deviation and percentile for BloodPressure, comparing the results using charts and repoting findings.   

### How to run
1. Put the original `diabetes.csv` in `data_raw/`
2. Open the notebook `Diabetes_Sampling_Bootstrap.ipynb`
3. Running all cells will automatically creates:
   - `data_clean/clean_data.csv`
   - `reports/bootstrap_summary.md`
