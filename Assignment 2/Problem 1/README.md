# Used Cars Price Analysis

### What the notebook does (all 5 tasks)
a) Missing values handling  
b) Remove units (kmpl, CC, bhp)  
c) One-hot encode Fuel_Type & Transmission  
d) New feature, Car_Age  
e) All pandas operations: select, filter, rename, mutate, arrange, groupby+summarize

### How to run
1. Put the original `train.csv` in `data_raw/`
2. Open the notebook `problem_1.ipynb`
3. Running all cells will automatically creates:
   - `data_clean/clean_data.csv`
   - `reports/summary.md` (top cities by average price)
